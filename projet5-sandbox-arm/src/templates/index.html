Endpoints REST
GET/api/status
Statut du sandbox, QEMU disponibles, compteurs
curl http://localhost:5000/api/status
GET/api/samples
Liste les binaires disponibles dans samples/
POST/api/analyze
Analyse un binaire — upload (multipart) ou fichier existant (JSON)
# Upload fichier curl -X POST -F "file=@mon_binaire" http://localhost:5000/api/analyze # Fichier dans samples/ curl -X POST -H "Content-Type: application/json" \ -d '{"filename":"malware_arm_bin"}' http://localhost:5000/api/analyze
POST/api/batch
Analyse plusieurs binaires en parallèle (max 10)
curl -X POST -H "Content-Type: application/json" \ -d '{"filenames":["bin1","bin2","bin3"]}' http://localhost:5000/api/batch
GET/api/jobs/<job_id>
Polling du statut d'un job (queued / running / done / error)
curl http://localhost:5000/api/jobs/a1b2c3d4
GET/api/reports
Liste tous les rapports avec score et MITRE
GET/api/reports/<name>
Rapport JSON complet d'un binaire analysé
curl http://localhost:5000/api/reports/malware_arm_bin
